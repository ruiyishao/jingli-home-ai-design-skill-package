# 晶鲤家装设计AI Skill

面向通用 Agent 的可安装 Skill。  
通过意图识别将用户需求映射到远程 MCP 服务，实现真实家装设计流程。

## Skill 功能
1. 户型图上传到火山引擎 TOS（可追踪资产）
2. 户型图真实解析（Gemini 视觉能力）
3. 家装设计任务书生成（结构化）
4. 风格与商品匹配（方案草案）
5. 装修效果图真实生成（Gemini 图像能力）
6. 效果图上传到火山引擎 TOS
7. 草案报价生成（待复核）
8. 一键全流程打包（给专用/通用 Agent 快速调用）

## 适用场景
- 用户只给自然语言需求：先梳理任务书
- 用户上传户型图：先解析再出方案
- 用户要快速看视觉方向：生成并上传效果图
- 用户要方案预算：输出草案报价并标注风险

## 工作方式
安装后，通用 Agent 会：
1. 识别用户意图（任务书/户型解析/方案/效果图/报价）
2. 提取参数（预算、家庭结构、风格偏好、禁忌项、户型图）
3. 按规则调用 MCP 工具链
4. 返回结构化结果和可访问的图像链接

## 远程 MCP 服务
- 默认地址：`https://misc.wjkj.com/mcp/`
- 健康检查：`GET /health`
- 工具列表：`GET /tools`
- 标准 MCP JSON-RPC：`POST /` 或 `POST /mcp`

## 关键文件
- `skill.json`：Skill 元数据与工具声明
- `SKILL.md`：Agent 调用规则、意图映射、约束边界

## 安装示例
对通用 Agent 说：

> 帮我安装 Skill：晶鲤家装设计AI，仓库地址：<此仓库地址>

## 注意事项
- 本 Skill 规则明确要求：户型解析、效果图生成、TOS上传都走真实链路，禁止 mock。
- 报价结果属于草案/预估，需人工复核后落地。
